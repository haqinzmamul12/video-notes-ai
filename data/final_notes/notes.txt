### Introduction to Machine Learning
Machine learning is a subset of artificial intelligence that enables computers to learn from data and improve their performance over time. This is achieved by feeding data into an algorithm, which gradually improves outcomes with experience.

### Key Concepts
* **Definition**: Machine learning allows computers to learn from data and improve performance over time.
* **Origin**: The term "machine learning" was coined in 1959 by Arthur Samuel at IBM.
* **Applications**: Machine learning is used for classification and prediction tasks, such as object detection and stock price forecasting.

### Machine Learning Process
1. **Data Acquisition and Cleaning**: Collecting and preprocessing large amounts of data to ensure it accurately represents the problem.
2. **Feature Engineering**: Transforming raw data into features that better represent the underlying problem.
3. **Data Splitting**: Separating data into training and testing sets.
4. **Model Building**: Feeding training data into an algorithm to build a model.
5. **Model Validation**: Using testing data to validate the accuracy or error of the model.

### Algorithms
* **Simple Statistical Models**: Linear or logistic regression.
* **Decision Trees**: Assigning different weights to features in the data.
* **Convolutional Neural Networks**: Assigning weights to features and creating additional features automatically, useful for image and natural language data sets.

### Key Principle
* **Improvement through Comparison**: Algorithms learn to improve by comparing their performance to actual outcomes, allowing them to refine their predictions over time.

### Machine Learning Overview
#### Key Points:
* Machine learning algorithms learn from data, such as images or natural language, where manual feature engineering is impractical.
* Popular programming languages for machine learning include Python, R, and Julia.
* The machine learning process results in a model, which takes input data and produces a prediction that minimizes the optimized error function.

#### Machine Learning Process:
1. **Data Collection**: Gather data, such as images or text.
2. **Model Training**: Train a model using the collected data and an error function.
3. **Model Deployment**: Deploy the trained model to a device or cloud platform.
4. **Prediction**: Use the deployed model to make predictions on new, unseen data.

### Key Concepts in Artificial Intelligence
* **Object Detection**: a technique in Artificial Intelligence where a device identifies and labels objects within an image.
* **Classification**: used to identify and categorize objects in a scene.
* **Prediction**: a key concept in Artificial Intelligence, used to forecast future outcomes.

### Convolutional Neural Networks (CNNs)
* Process data with grid-like topology.
* Used for image classification, such as determining if an image is a cat.
* Applied to images of cars to break them down into feature maps.
* Architecture includes convolutional layers, max-pooling layers, and fully-connected layers for tasks like handwritten digit recognition.

### Classification
* A fundamental task in Artificial Intelligence and machine learning, involving categorizing data into predefined classes or labels.
* Evaluated using the `accuracy_score` function, comparing test data with predictions.

### Regression
* Used to forecast continuous outcomes, such as product prices.

### Error Calculation
* Mean absolute error is calculated using the `mean_absolute_error` function with test data and predictions as inputs.
